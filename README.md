# React Dynamic Search
Dynamic search application with React


## Tech Stack, Lib, Tools
`React`, `Axios`, `lodash`, `MUI`


## Run the app
```
npm start
npm test
```

## Run on Production
```
npm run build
```

## Configuration
`.env`
```
REACT_APP_API_URL="http://localhost:4000/users"
```
